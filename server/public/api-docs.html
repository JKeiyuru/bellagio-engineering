Bellagio Engineering API Documentation

This documentation provides information on the available endpoints for the Bellagio Engineering API.

Authentication

For protected endpoints, include a Bearer token in the Authorization header: Authorization: Bearer your_token_here

POST /api/admin/login

Login as admin and get authentication token

Request Body:
{
  "email": "admin@example.com",
  "password": "password123"
}

Projects

GET /api/projects

Get all projects with pagination

Query Parameters:
Parameter Type Description
page Number Page number (default: 1)
limit Number Number of items per page (default: 10)
market String Filter by market ID (optional)
GET /api/projects/:id

Get a specific project by ID

POST /api/projects

Create a new project (Admin only)

Request Body:
{
  "title": "Project Title",
  "description": "Project description",
  "location": "Nairobi",
  "client": "Client Name",
  "startDate": "2023-01-01",
  "endDate": "2023-06-30",
  "market": "market_id_here",
  "services": ["Service 1", "Service 2"],
  "featured": true
}
PUT /api/projects/:id

Update a project (Admin only)

DELETE /api/projects/:id

Delete a project (Admin only)
